# Dream Tycoon VR - Assets

This folder contains game assets.

## Audio Files Needed

Place the following audio files in `sounds/`:

| File | Description | Recommended |
|------|-------------|-------------|
| `select.mp3` | Button click/selection | Short click sound |
| `win.mp3` | Win celebration | Upbeat chime or fanfare |
| `loss.mp3` | Loss feedback | Subtle low tone |
| `spin.mp3` | Market spinning | Slot machine spin |
| `ambient.mp3` | Background music | Casino/trading floor ambience |

## Free Sound Resources

- [Freesound.org](https://freesound.org/)
- [Mixkit](https://mixkit.co/free-sound-effects/)
- [Pixabay](https://pixabay.com/sound-effects/)

## 3D Models (Optional)

Place GLTF models in `models/` for future enhancements:
- `slot_machine.glb` - Detailed slot machine
- `coin.glb` - Gold coin model
- `trophy.glb` - Win trophy
